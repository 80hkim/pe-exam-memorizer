# 자율운항선박(MASS)

## 1. 개요

1) 인공지능(AI)과 디지털 항법을 활용한 차세대 선박 자율운항 체계임

## 2. 핵심 개념 및 특징

가. 운항 체계

• 센서·통신·AI 기반의 자동 인지·판단·조종을 수행하는 선박 체계임
• 원격지원→부분자율→완전자율의 단계별 발전 구조로 구분됨

나. 기술 특징

• 선박-육상 간 실시간 통신 연동으로 운항 데이터를 공유함
• 자율 충돌회피 및 경로 최적화 알고리즘으로 안전 항행을 지원함

## 3. 기능·의의 / 적용 유의점

가. 항만 수역시설 영향

• 해상교통관제(VTS) 기능 고도화 및 자율선박 전용 관제체계를 구축함
• 선박조종 시뮬레이션을 통한 항로·선회장 적정 규모를 검증함

나. 적용 유의점

• 통신 신뢰성, 사이버 보안, 책임소재 등 법·제도 기반을 정비함
• IMO MASS Code 등 국제기준과의 정합성을 확보함

## 4. 결론

1) 항만 수역시설 및 관제기준 재정립이 자율운항선박 도입의 선결 과제임
